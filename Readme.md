# Spell Checker for WinForms - Check Spelling to a Text Box

This example illustrates how to load multiple dictionaries to check the spelling of a multi-language text in a simple text box control. Also, you will see how to addÂ a Custom Dictionary so that the user can add custom words to it.

Make sure that the "Copy to Output Directory" option for the dictionary files is set to "If Newer". This action prevents custom dictionary files from being overwritten.
